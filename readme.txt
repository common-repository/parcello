The Parcello plugin connects your wordpress with the parcello api.

== Description ==

This Plugin will enable you to manage your Shipments on parcello.

It will automatically sync your shipments wirh parcello.

By using this Plugin we have to connect with the Parcello API so Plugins data will be shared with Parcello.

We know that your Data is sensible and you are always able to remove all of your Data by removing this Plugin and deletion of your Parcello account.

Feedback is greatly appreciated and we would love to hear from you via the plugin's support and features page.
